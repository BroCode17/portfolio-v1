| Breakpoint | Description |
| ---------- | ----------- |
| < 481 | Mobile devices |
| 481 - 768px | iPads, Tables |
| 769 - 1024px | small screens, laptops |
| 1024 - 1200px | Desktops, laprge Screens | 
| 1201 and greater | Extra large screens, TV|
